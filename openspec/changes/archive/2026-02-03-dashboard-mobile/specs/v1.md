# Specs: Mobile Dashboard v1

## Requirements
1.  **Viewport**: Application MUST render correctly on width 375px (iPhone SE) and up.
2.  **Navigation**:
    - Sidebar MUST be hidden by default on mobile.
    - User MUST be able to open Sidebar via a Menu button.
    - User MUST be able to close Sidebar.
3.  **Data Visualization**:
    - KPI Cards MUST stack vertically on mobile.
    - Calendar MUST default to `listWeek` (List View) on mobile to avoid horizontal scrolling/squashed columns.
4.  **Modals**:
    - "Nova Consulta" modal MUST have `width: 90%` and suitable margins on mobile.

## Technical Constraints
- No new framework (Bootstrap/Tailwind). Use existing raw CSS.
- Preserved desktop experience.
