# Specs: Ngrok Container v1

## Requirements
1.  **Docker Service**: The application stack MUST include an `ngrok` service.
2.  **Connectivity**: The `ngrok` container MUST tunnel HTTP traffic to the `app` container on port 8000.
3.  **Authentication**: The service MUST use `NGROK_AUTHTOKEN` from the environment.
4.  **Visibility**: The user MUST be able to see the public URL in the container logs.

## Configurations
- **Image**: `ngrok/ngrok:latest`
- **Network**: Same bridge network as `app`.
